TCP Web Server Implementation

This project demonstrates:

  • TCP Socket Programming: Direct socket communication
  • HTTP Protocol: Manual HTTP request/response handling
  • Threading: Concurrent client handling
  • File System Operations: Static file serving
  • Security: Path traversal protection

Supported Features:

  • GET requests only
  • HTML, CSS, and JavaScript files
  • Proper MIME types
  • HTTP error codes (404, 405, 403)
